This file contains microsatellite data of Testudo hermanni in Europe.

The data are constituted by 9 microsatellite markers.
Each locus is encoded in the three digit format (116116).
Each column corresponds to a locus specified in the order at the beginning of the file.

Pop indicates the beginning of a new population.

BM = Bosco Mesola population (Italy)
BN = Bosco Nordio population (Italy)
ITCS = Central and Southern Italian population (Italy)
SICO = Sicilian and Corsican population (Italy, France)
SP = Spain Population (Spain)
CROA = Croatian poputation (Croatia)
FR = French population (France)
GR = Greek population (Greece)
MAC = Macedonian population (Macedonia)
